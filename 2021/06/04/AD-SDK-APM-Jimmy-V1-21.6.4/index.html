性能指标

广告 SDK 性能指标：

  • 网络耗时
    • HTTPDNS 耗时
    • HTTP Request & Response 耗时
    • Image download 耗时
    • Video download 耗时
  • 加密耗时
    • AES加密耗时
    • RSA加密耗时
  • 页面渲染（待定）
    • 文件类渲染处理耗时

广告 SDK 业务指标：

  • 广告请求计数
  • 广告响应计数
  • 广告填充计数
  • 广告曝光计数
  • 广告点击计数
  • 广告点击后续动作
    • 后续动作类型（打开落地页，下载，跳转应用）
  • 错误日志
    • 响应超时，失败等异常场景
    • 曝光异常统计（异常原因/会话 id）
    • 点击异常统计（异常原因/会话 id）

具体流程（iOS端）

统计上传时机类型

说明：上传机制有不同类型，确保适合不同使用场景。

统计类型

需统计的类型，划分为各个独立的小项。其中 广告点击后续动作 和 错误日志 需自定义状态码。

数据模型

耗时存放于数组之中，计数为 String 类型，自定义的广告点击后续动作和错误日志存放于数组中。
